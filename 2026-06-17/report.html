計算方式說明

Alpha（超額報酬）
alpha = 個股報酬率 − 0050 報酬率
扣除大盤（0050）漲跌後的相對表現。正值代表跑贏大盤，負值代表跑輸大盤。 假設各股 Beta = 1，為簡化版超額報酬計算。

時間窗口
1日 / 5日 / 10日 / 20日 均為交易日，已自動排除週末與假日。

Ridge 迴歸係數（超額報酬貢獻）
以標籤矩陣（one-hot）為 X、各股 alpha 為 Y，跑橫截面 Ridge 迴歸。 係數已換算為百分比，例如 +2.00% 代表：排除其他族群重疊後， 單獨因為這個標籤，在該時間窗口內相對大盤多漲約 2%。
正值 = 跑贏大盤；負值 = 跑輸大盤。
⚠ Ridge 正則化會將係數往 0 壓縮，真實效應可能略大於顯示值。

已知限制：Beta 假設為 1 可能高估高 Beta 股的 alpha； 1日 alpha 雜訊較大，建議以 5日 / 20日 為主要參考。
